📖 How to use this workout program builder

Method 1 – Smart Import (easiest):
1. Go to the Smart Import tab
2. Paste your workout program in the format shown above
3. Click "Import Workouts & Schedule" – everything is created automatically

Method 2 – Manual:
1. Create workouts in Workout Library tab
2. Set weekly schedule in Weekly Schedule tab
3. Click "Apply Schedule to Calendar"

Using the Calendar:
- Click any date to see scheduled workouts
- Check off exercises as you complete them
- Click "Complete This Workout" to mark the whole workout done
- Weekly progress (e.g., 2/3) updates automatically
