📖 How to use this tracker

Today's Workout – Shows all workouts scheduled for today. For each exercise you can:
• ✅ Check the box when you complete an exercise
• 🕐 Click "Record Rep" to timestamp each rep (shows exact time with milliseconds)
• ⏲️ Use the timer (set minutes/seconds, Start/Pause/Stop/Reset)

Calendar – Click any date to see workouts for that day below the calendar. Green dates have scheduled workouts.

Import – Paste your program in the format shown. The import automatically creates workouts AND weekly frequencies.

Weekly Progress – Shows how many times you've completed each workout this week (e.g., 2/3).
