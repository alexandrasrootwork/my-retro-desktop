📖 WORKOUT PROGRAM FORMAT SPECIFICATION

RULE 1: Workout Definition
• Each workout starts with Workout Name: [name] on its own line
• Each exercise starts with - (dash + space)
• Exercise format: name: X sets of Y reps or name: X sets of Y sec
• Optional video: add !video(URL) at the end
• Separate workouts with a blank line

RULE 2: Weekly Schedule
• Write Weekly schedule: on its own line
• Each week: Week 1: WorkoutName 3x/week, WorkoutName 2x/week
• No spaces between number and "x"
• Comma + space between multiple workouts
• Workout names must match EXACTLY the names above
