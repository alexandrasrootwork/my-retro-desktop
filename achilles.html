📋 Workout Library
📆 Weekly Schedule
❓ Help
📅 Select a date
💡 Create workouts with exercises. Each exercise can have sets, reps, and a video link.
💡 For each week of your program, set how many times per week each workout should be done.
📖 How to use this workout program builder

1. Create Workouts: Go to Workout Library → Add Workout. Give it a name, then add exercises with sets/reps/video links.

2. Set Weekly Schedule: Go to Weekly Schedule → Add Week. For each workout, set how many times per week (e.g., Workout A: 3x/week).

3. Apply Schedule: Click "Apply Schedule to Calendar" – this distributes workouts across the days of each week.

4. Daily Logging: Click any date on the calendar. Check off exercises as you complete them. Click "Complete This Workout" to mark the whole workout done for the day.

5. Weekly Progress: The calendar shows workout completion. You can do extra workouts on any day even if not scheduled.

🤖 For AI / ChatGPT: Ask me to create workouts in this format:
Workout Name: Leg Day
- Squats: 3 sets of 10 reps
- Calf raises: 3 sets of 15 reps !video(URL)
- Leg lifts: 3 sets of 12 reps

Then specify weekly schedule like: "Week 1: Leg Day 2x/week, Upper Day 2x/week"
🏋️ Click calendar to log workouts Local only
